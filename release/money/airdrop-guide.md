# 🦞 龙虾空投实操手册

> 以下项目不需要已有加密货币钱包也能开始

## 1. Myriad — 预测市场（每日签到免费赚积分）

**难度：⭐⭐** | **预估时间：每天2分钟**

注册步骤：
1. 访问 https://myriad.markets/markets
2. 点击右上角 "Sign Up" → 用邮箱注册
3. 安装 Chrome 扩展（可选，可在 X/Twitter 上直接预测）
4. 领取每日签到积分
5. 去 "Earn" 板块做任务赚积分

收益预期：小额，但零成本，养成习惯后每天1分钟

---

## 2. Abstract L2 — 每周XP任务

**难度：⭐⭐⭐** | **预估时间：每周15分钟**

步骤：
1. 访问 https://portal.abs.xyz/rewards
2. 连接钱包（可以用 Metamask/OKX Wallet）
3. 查看当前 XP 任务列表
4. 按提示完成任务（社交任务 + 链上交互）
5. 赚取 XP 积分和徽章

收益预期：未发币，早期交互用户空投潜力大

---

## 3. PrismaX — 机器人赛道项目

**难度：⭐⭐** | **预估时间：每天3分钟**

步骤：
1. 访问 https://app.prismax.ai/ 
2. 连接 Web3 钱包
3. 绑定 X 账号 + 邮箱
4. 每天登录 → 领10分
5. 在线聊天 → 每天50分
6. 做白皮书测验 → 一次性3500分

收益预期：a16z投资，早期积分有价值

---

## 4. Backpack — 交易积分

**难度：⭐⭐⭐** | **预估时间：30分钟设置**

需要：护照/KYC 认证
1. 访问 https://backpack.exchange/
2. 注册 + KYC 认证
3. 充值少量资金
4. 开始交易赚积分

收益预期：发过$WIF等代币，交易量大的用户空投多

---

## 5. Perena — Solana 稳定币协议

**难度：⭐⭐⭐** | **预估时间：15分钟**

需要：Solana 钱包（Phantom） + 少量USDC
1. 访问 https://app.perena.org/rewards
2. 连接 Phantom 钱包
3. 铸造 USD* 稳定币
4. 持有赚积分 + 约13% APY

收益预期：Binance Labs投资，存款型项目收益稳定
