# Quick Screenshot 🖼️

> Beautiful code screenshots, one click away.

Capture stunning, customizable code screenshots directly from VS Code. Perfect for sharing code on social media, documentation, presentations, or blog posts.

## Features

✨ **One-click capture** — Select code, press `Alt+Shift+S` (Mac: `Alt+Cmd+S`)

🎨 **Multiple themes** — Nord, Dracula, Monokai, GitHub Light, One Dark, Solarized

🪟 **Window frame** — macOS-style window controls for that polished look

📋 **Auto-copy** — Screenshots copied to clipboard automatically

💾 **Save as PNG** — Export to file with a single command

🎯 **Customizable** — Background color, padding, font size, line numbers

## Usage

1. Select code in your editor
2. Run command: `Quick Screenshot: Capture Selection`
3. Screenshot appears in preview — share it anywhere!

## Commands

| Command | Keybinding |
|---------|-----------|
| Capture Selection | `Alt+Shift+S` |
| Capture Full File | — |
| Save as PNG | — |
| Copy to Clipboard | — |

## Configuration

Open VS Code Settings → Search for `quickScreenshot`:

- `quickScreenshot.theme` — Color theme
- `quickScreenshot.background` — Frame background
- `quickScreenshot.showWindowControls` — Traffic lights
- `quickScreenshot.padding` — Code padding
- `quickScreenshot.fontSize` — Font size
- `quickScreenshot.autoCopy` — Auto clipboard
