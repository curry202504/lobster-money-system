## 一、4月最大新闻：OpenAI 关停 Sora

2026年4月，OpenAI 正式关停了上线仅6个月的 Sora。
- **日均运营成本：** 约 1500 万美元
- **关停原因：** 用户留存率断崖下跌，商业化难以为继

> Sora 的退场标志着 AI 视频从"拼参数"进入"拼落地"阶段。

## 四、零基础 AI 视频制作教程

### 工具推荐

|------|------|----------|
| **Runway Gen-4.5** | 订阅制 $12-95/月 | 影视级制作，精确控制 |
| **Pika Labs** | 免费+付费 | 快速原型，创意实验 |
| **CapCut/剪映** | 免费 | 短视频+AI特效，最容易上手 |
| **HeyGen** | 免费试用 | AI数字人播报视频 |

#### 国内工具（适合中文内容，国内可用）
| 工具 | 费用 | 特点 |
|------|------|------|
| **可灵 3.0（快手）** | API/订阅 | 4K输出，15秒，国内最成熟 |
| **Seedance/即梦（字节）** | API | 最长60秒，CapCut联动 |
| **文升智链** | ~30元/分钟 | AI短剧全流程自动化 |
| **SkyReels** | 等待商用 | 双流音视频生成 |

### AI 短剧制作四步法

#### 第一步：准备剧本
- 新手推荐题材：**爽文逆袭、悬疑推理、甜宠恋爱**
- 来源：自己写 / 改编爆款网文

#### 第二步：选择工具并导入
- **快速上手推荐：** 剪映 + AI 功能
  1. 打开剪映 → AI 创作 → 智能生成脚本
  2. 输入剧本大纲
  3. 自动分镜 + 配音 + BGM
- **进阶推荐：** 可灵 3.0
  1. 注册快手开放平台获取 API
  2. 用提示词描述场景
  3. 逐镜生成后剪辑合成

#### 第三步：微调优化
- 调整画风（24种可选）
- 选择配音音色
- 调整镜头顺序和时长
- 添加字幕

#### 第四步：导出与发布
- **国内平台：** 抖音、快手、B站、视频号
- **海外平台：** TikTok、YouTube Shorts、Instagram Reels
- **合规要求：** 标注"AI生成"，4月起需备案

### 变现路径（6种）

| 方式 | 说明 | 收益 |
|------|------|------|
| 1️⃣ **平台流量分成** | 按播放量赚钱 | 万次播放20-200元 |
| 2️⃣ **付费解锁分账** | 前几集免费，后面付费 | 创作者拿70-75% |
| 3️⃣ **小说分销** | 挂载小说小程序赚佣金 | 佣金20-50% |
| 4️⃣ **B端商单** | 品牌定制植入 | 500-5000元/条 |
| 5️⃣ **私域闭环** | 引流到微信卖产品/课程 | 利润更高 |
| 6️⃣ **出海赚美金** | 翻译后发海外平台 | 单价更高 |

## 五、新手避坑指南

1. **别追求完美主义** — AI短剧是快消品，速度 > 质量
2. **先备案再上线** — 4月起必须合规
3. **别单打独斗太久** — 跑通模式后尽快组建团队
4. **工具别用错** — 免费工具崩脸、画质差，建议用主流付费工具
5. **选题决定生死** — 爽文/悬疑/甜宠出爆款概率最高
