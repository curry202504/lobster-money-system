## 二、实操教程：可灵 3.0 从零开始

### 第一步：注册登录
1. 打开 https://app.klingai.com
2. 手机号/微信注册
3. 登录后首页能看到精选视频作品

### 第二步：文生视频（Text-to-Video）
1. 首页点击 **"AI视频" → "文生视频"**
2. **选择模型：** 选最新的 **可灵 3.0**
3. **写提示词（Prompt）：** 详细描述画面
   - ✅ 正确写法：`一只橘猫在咖啡店窗台上晒太阳，午后阳光照在它的毛上，4K画质，电影感`
   - ❌ 错误写法：`一只猫`
4. **设置参数：**
   - 时长：5秒 / 10秒 / 15秒
   - 分辨率：1080P / 4K
   - 比例：16:9（横屏） / 9:16（竖屏/抖音）
5. 点击 **生成**，消耗灵感值
6. 等待 1-3 分钟，查看结果

### 第三步：图生视频（Image-to-Video）
1. 选择 **"图生视频"**
2. **上传参考图：** 支持单张图片或两张（首尾帧）
3. **写提示词：** 描述图片中要发生的动作
   - 例：`海浪缓缓拍打礁石，水花四溅，慢动作`
4. 点击生成

### 第四步：动作控制（⭐ 4月最新功能）
这是可灵 3.0 最炸裂的功能，**让一张照片做出指定动作**：

1. 进入 **"动作控制"**
2. **上传参考图：** 一张人物照片
3. **上传动作视频：** 一段你想让照片模仿的动作视频
4. **绑定面部主体：** 勾选后能保持面部一致性
5. **填写提示词：** 补充特殊要求
6. 点击 **生成**
7. 效果：照片中的人物会一比一复刻动作视频里的动作

### 第五步：利用模板一键同款
- 在首页浏览别人生成的作品
- 看到喜欢的点击 **"一键同款"**
- 直接跳转到生成界面，替换成你自己的素材

## 四、快速上手技巧

### 提示词怎么写？
| 要素 | 说明 | 示例 |
|------|------|------|
| 主体 | 谁/什么 | 一只橘猫 |
| 动作 | 在做什么 | 在窗台上晒太阳 |
| 环境 | 在哪里 | 咖啡店窗台 |
| 光影 | 光线效果 | 午后阳光，暖色调 |
| 镜头 | 拍摄方式 | 特写，电影感，浅景深 |
| 画质 | 质量标准 | 4K，高细节，写实 |

### 常见参数设置
- **种子（Seed）：** 固定后每次生成同一风格
- **CFG Scale：** 7-12（越高越贴近提示词）
- **步数（Steps）：** 30-50（越多细节越丰富）
- **负面提示词：** 告诉 AI 不要出现什么（例：`畸变, 崩脸, 多指`）

### 新手避坑
1. **提示词越详细越好** — 3-5个完整句子
2. **不要用免费山寨工具** — 画质差，崩脸严重
3. **图生视频比文生视频更容易控制**
4. **先用模板一键同款**，熟悉流程后再自己写提示词
5. **4月起国内平台要求标注"AI生成"**

4. 导出发布

- 可灵支持数字人生成
- 上传一段视频创建面部绑定
- 用于口播、讲解类视频

### 3. 多镜头叙事
- PixVerse V6 支持镜头切换
- 切换时光线、材质、角色保持一致
