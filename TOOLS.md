# TOOLS.md - Local Notes

Skills define _how_ tools work. This file is for _your_ specifics — the stuff that's unique to your setup.

## What Goes Here

Things like:

- Camera names and locations
- SSH hosts and aliases
- Preferred voices for TTS
- Speaker/room names
- Device nicknames
- Anything environment-specific

## Examples

```markdown
### Cameras

- living-room → Main area, 180° wide angle
- front-door → Entrance, motion-triggered

### SSH

```

## Why Separate?

Skills are shared. Your setup is yours. Keeping them apart means you can update skills without losing your notes, and share skills without leaking your infrastructure.

## Industrial Research Gotchas

### 高空幕墙清洗机器人 (Curtain Wall Cleaning Robot) Research
- **Amazon does NOT sell industrial grade** ($100k+) — only consumer window robots ($100-600)
- Industrial products are B2B: Alibaba, brand websites, B2B platforms (gys.cn, made-in-china)
- Use 360搜索 (so.com) for Chinese industrial data (accessible, no captcha)
- SERBOT Chinese distributor: 烟台海特林 (serbot.com.cn)
- Key industrial prices: SERBOT I ¥264,000, SERBOT III ¥300,000, 哈工鹏泽GE02 ~¥338,000

### Screenshots
- Screenshot actual product/website pages, NOT search results
- Always check file size (>50KB for meaningful web page)
- **VERIFY: after taking screenshot, check the file visually (image analysis) to confirm it shows the right content, not a blank/error page**
- File size < 30KB usually means blank/error — re-take
- For SSL errors: click "高级" → "继续前往" to bypass
- Self-validation loop: screenshot → verify → fix if needed → screenshot again
